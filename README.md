# 🖼️ Image Filter App

A simple Python application that applies different image filters using OpenCV. Users can choose a filter from the console menu, and the processed image is displayed and saved automatically.

---

## 📌 Features

- 🎨 Convert image to Grayscale
- 🌫️ Apply Blur Effect
- 🖌️ Cartoon Filter
- 💾 Save filtered images
- 🖥️ Console-based menu interface

---

## 🛠️ Technologies Used

- Python 3
- OpenCV (cv2)
- NumPy

## 🚀 Installation

1. Clone this repository:

```bash
git clone https://github.com/your-username/image-filter-app.git
```

2. Navigate to the project folder:

```bash
cd image-filter-app
```

3. Install the required libraries:

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Project

```bash
python image_filter.py
```
